# 🍕 Pizza Sales Dashboard

## 📊 Overview
This project analyzes pizza sales data using Power BI to generate business insights.

## 🔍 Key Insights
- Top Revenue Product: thai_ckn_l
- Top Ordered Product: big_meat_s
- Top Selling Category: Classic
- Peak Hour: 12 PM

## 📈 Features
- Sales by Hour Analysis
- Orders by Day
- Category Filter
- KPI Cards (Total Sales, AOV, Orders)

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
